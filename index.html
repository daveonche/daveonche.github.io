Welcome to GIT!!

Git Clone

Clone a repository to your local pc git clone repo_url/repo_ssh

Git Add

Add specific changes git add filename

Add all changes git add .

Git Commit

Save/commit changes git commit -m "message"

Add and commit changes at once git commit -am "message"

Git Push

Push changes to github repo git push

Git Merge

Merge conflicts from two different changes on the same line. Fix the conflicts git merge

Git Log

log all commits on the repo git log

Git Reset

Reset the current changes to previous commit git reset --hard commit_hash

git reset --hard origin/master

Create branches to add features and fix bugs simultaneously without disrupting the master version

Git Branching

See what branches are on the repo git branch

Create new branch git checkout -b branch_name

Switch branch git checkout branch_name

switch to the master or main branch and merge with changes from the new branch once satisfied

Git Fork

You can make copies of original repos to your own repo and clone to your local pc. Make changes and also contribute by making pull requests to the original repo
